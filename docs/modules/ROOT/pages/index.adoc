= RuboCop Vicenzo
:toc: left

Custom RuboCop cops for enforcing conventions adopted by Vicenzo projects.

== Installation

Add to your `Gemfile`:

[source,ruby]
----
gem 'rubocop-vicenzo', require: false
----

Then add to your `.rubocop.yml`:

[source,yaml]
----
plugins:
  - rubocop-vicenzo
----
